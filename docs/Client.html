schema(measurement, schema, options)

Source:
  • lib/client.js, line 780
Since:
  • 2.4.0

Set or Get schema for the measurement

Examples
const schema = {
  use: 'integer',
  sucesss: 'boolean',
  vip: 'boolean',
};
client.schema('request', schema);
client.write('request')
  .field({
    use: 300,
    sucesss: 'T',
    vip: 'true',
  })
  .then(() => {
    console.info('write point success');
  })
  .catch(console.error);
const schema = {
  use: 'integer',
  sucesss: 'boolean',
  vip: 'boolean',
};
client.schema('request', schema, {
  stripUnknown: true,
});
client.write('request')
  .field({
    use: 300,
    sucesss: 'T',
    vip: 'true',
    account: 'vicanso',
  })
  .then(() => {
    console.info('write point success');
  })
  .catch(console.error);
Parameters:
Name Type Description
measurement String

the name of measurement
schema Object

the schema definition, if is undefined, will return the schme for the measurement
options Object

the schema options, { stripUnknown: Boolean, // remove all unknown field }
